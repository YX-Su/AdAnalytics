# 广告点击实时分析系统

模拟互联网广告投放平台的完整数据链路，从前端埋点到数仓分层到可视化报表。

🔗 **在线演示**：https://你的用户名.github.io/ad-analytics/

## 技术栈

- **Kafka** — 消息队列，解耦数据采集与计算层
- **Spark Structured Streaming** — micro-batch 实时计算 PV、UV、CTR
- **数仓四层分层** — ODS 原始存储 → DWD 清洗过滤 → DWS 聚合宽表 → ADS 报表输出
- **双表 JOIN 分析** — 点击日志表 × 广告维度表，支持品类维度 CTR 分析和预算利用率计算

## 核心功能

- 前端广告信息流模拟器，可手动点击/滑动触发埋点事件
- 一键启动 15 秒自动压测，模拟多用户并发场景
- 实时 Kafka 消费日志、Spark 处理延迟监控
- 数据表导入分析（CSV 粘贴 → JOIN → 多维度报表输出）
- 数仓四层实时联动可视化

## 使用方法

打开 `index.html` 即可，无需任何后端依赖。

## 项目结构

纯前端实现，单文件 HTML，使用原生 JavaScript + Canvas，无第三方框架。
